DebugClient
===========

**Inherits:** `Control <https://docs.godotengine.org/en/stable/classes/class_control.html>`__

A world-space nameplate for visualizing player network identity and authority.

.. rst-class:: classref-introduction-group

Description
-----------

Automatically tracks a :ref:`SpawnerComponent<class_SpawnerComponent>` or ``Node`` and displays its `multiplayer.get_unique_id <https://docs.godotengine.org/en/stable/classes/class_multiplayer.html#class_multiplayer_property_get_unique_id>`__ and username in world-space using unprojection.

AuthCoordinator
===============

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Internal coordinator for :ref:`MultiplayerTree<class_MultiplayerTree>` authentication hooks.

.. rst-class:: classref-introduction-group

Description
-----------

Binds ``SceneMultiplayer`` auth callbacks to a :ref:`NetwAuthProvider<class_NetwAuthProvider>` and stores accepted identities or rejection reasons in :ref:`SessionRoster<class_SessionRoster>`.

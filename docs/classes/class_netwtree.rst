NetwTree
========

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Thin facade over :ref:`MultiplayerTree<class_MultiplayerTree>` exposing only session-level APIs.

.. rst-class:: classref-introduction-group

Description
-----------

Components obtain this via :ref:`NetwContext.tree<class_NetwContext_property_tree>` rather than holding a direct reference to :ref:`MultiplayerTree<class_MultiplayerTree>`. This keeps component code off the concrete :ref:`MultiplayerTree<class_MultiplayerTree>` class (backend, multiplayer_api, etc.).  



Holds a ``WeakRef`` so components that cache an instance survive tree teardown without keeping the :ref:`MultiplayerTree<class_MultiplayerTree>` alive.

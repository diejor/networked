NetDebugTreeContext
===================

**Inherits:** `Node <https://docs.godotengine.org/en/stable/classes/class_node.html>`__

Orchestrates debug signals and visuals for a single :ref:`MultiplayerTree<class_MultiplayerTree>` instance.

.. rst-class:: classref-introduction-group

Description
-----------

Owns the full lifecycle of all per-tree debug connections: scene events, per-scene synchronizer hooks, peer events, and visual decorations (nameplates). 



State is isolated to this instance to support multi-window Embedded Server.

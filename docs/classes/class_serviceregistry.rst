ServiceRegistry
===============

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Internal registry for :ref:`MultiplayerTree<class_MultiplayerTree>` session services.

.. rst-class:: classref-introduction-group

Description
-----------

Stores service nodes by script type while :ref:`MultiplayerTree<class_MultiplayerTree>` remains the public API surface and owner of node-tree searches.

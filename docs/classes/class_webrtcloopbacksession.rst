WebRTCLoopbackSession
=====================

**Inherits:** `Resource <https://docs.godotengine.org/en/stable/classes/class_resource.html>`__

In-process WebRTC loopback session used by :ref:`WebRTCLoopbackBackend<class_WebRTCLoopbackBackend>`.

.. rst-class:: classref-introduction-group

Description
-----------

Performs the full WebRTC offer/answer/ICE handshake in-memory between a server ``WebRTCMultiplayerPeer`` and a client ``WebRTCMultiplayerPeer``, with no real network.

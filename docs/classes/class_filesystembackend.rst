FileSystemBackend
=================

**Inherits:** :ref:`NetwBackend<class_NetwBackend>` **<** `Resource <https://docs.godotengine.org/en/stable/classes/class_resource.html>`__

:ref:`NetwBackend<class_NetwBackend>` that stores records as :ref:`DictionaryEntity<class_DictionaryEntity>` files on disk.

.. rst-class:: classref-introduction-group

Description
-----------

Records are written to ``<base_dir>/<table>/<id><extension>``. By default files use the compact binary ``.dict`` format. Set :ref:`use_text_format<class_FileSystemBackend_property_use_text_format>` to ``true`` to write human-readable ``.tdict`` JSON instead (useful during development).  **Ghost-table detection:** on :ref:`initialize()<class_FileSystemBackend_method_initialize>` any subdirectory found under :ref:`base_dir<class_FileSystemBackend_property_base_dir>` that is *not* present in the supplied schema is reported as a ghost table via `@GlobalScope.push_warning() <https://docs.godotengine.org/en/stable/classes/class_@globalscope.html#class_@GlobalScope_method_push_warning>`__. No data is ever deleted automatically.

DictionaryBackend
=================

**Inherits:** :ref:`NetwBackend<class_NetwBackend>` **<** `Resource <https://docs.godotengine.org/en/stable/classes/class_resource.html>`__

In-memory :ref:`NetwBackend<class_NetwBackend>` that stores records in a nested ``Dictionary``.

.. rst-class:: classref-introduction-group

Description
-----------

Ideal for unit testing :ref:`SaveComponent<class_SaveComponent>` and other systems that depend on :ref:`NetwDatabase<class_NetwDatabase>` without wanting the performance overhead or test-isolation flakiness of disk I/O.  **Note:** Data is lost when the node or resource holding this backend is freed.

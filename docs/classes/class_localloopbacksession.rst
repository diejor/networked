LocalLoopbackSession
====================

**Inherits:** `Resource <https://docs.godotengine.org/en/stable/classes/class_resource.html>`__

Shared in-process session that links :ref:`LocalMultiplayerPeer<class_LocalMultiplayerPeer>` instances without real sockets.

.. rst-class:: classref-introduction-group

Description
-----------

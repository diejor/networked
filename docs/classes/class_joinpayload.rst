JoinPayload
===========

**Inherits:** :ref:`Serde<class_Serde>` **<** `Resource <https://docs.godotengine.org/en/stable/classes/class_resource.html>`__

Serializable data bag describing a player attempting to connect to a session.

.. rst-class:: classref-introduction-group

Description
-----------

Pass a populated instance to :ref:`MultiplayerTree.connect_player()<class_MultiplayerTree_method_connect_player>` to authenticate and spawn a player, or serialize it for transmission via :ref:`serialize()<class_JoinPayload_method_serialize>`.

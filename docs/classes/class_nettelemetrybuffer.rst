NetTelemetryBuffer
==================

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Game-side ring buffer for pre-failure telemetry.

.. rst-class:: classref-introduction-group

Description
-----------

Records one entry per reporter flush cycle. When a failure is detected (race condition, C++ error), :ref:`freeze()<class_NetTelemetryBuffer_method_freeze>` stops the write pointer so :ref:`snapshot()<class_NetTelemetryBuffer_method_snapshot>` can return the historical slice leading to the crash.

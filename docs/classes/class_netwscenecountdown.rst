NetwSceneCountdown
==================

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Server-driven countdown that ticks once per second.

.. rst-class:: classref-introduction-group

Description
-----------

Obtain via :ref:`NetwScene.start_countdown()<class_NetwScene_method_start_countdown>` - do not construct directly. Clients do not receive a return value; they listen to `NetwScene.countdown_started <https://docs.godotengine.org/en/stable/classes/class_netwscene.html#class_NetwScene_signal_countdown_started>`__ and the subsequent `NetwScene.countdown_tick <https://docs.godotengine.org/en/stable/classes/class_netwscene.html#class_NetwScene_signal_countdown_tick>`__ / `NetwScene.countdown_finished <https://docs.godotengine.org/en/stable/classes/class_netwscene.html#class_NetwScene_signal_countdown_finished>`__ signals, which are broadcast automatically.

::

    # Server:
    var cd := ctx.scene.start_countdown(10)
    await cd.finished
    start_match()

    # Client (connect before the server starts the countdown):
    ctx.scene.countdown_started.connect(func(n): $Timer.text = str(n))
    ctx.scene.countdown_tick.connect(func(n): $Timer.text = str(n))
    ctx.scene.countdown_finished.connect(start_match)

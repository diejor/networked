NetwIdentity
============

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Validated player identity produced by an auth provider.

.. rst-class:: classref-introduction-group

Description
-----------

Returned by `NetwAuthProvider._authenticate() <https://docs.godotengine.org/en/stable/classes/class_netwauthprovider.html#class_NetwAuthProvider_private_method__authenticate>`__ and stored in :ref:`NetwIdentityBucket<class_NetwIdentityBucket>` on the server. The :ref:`username<class_NetwIdentity_property_username>` field is server-authoritative -- :ref:`MultiplayerTree.request_join_player()<class_MultiplayerTree_method_request_join_player>` uses it to override the client-claimed username.

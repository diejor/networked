`bool <https://docs.godotengine.org/en/stable/classes/class_bool.html>`__ **timeout**\ (\ target_signal\: `Signal <https://docs.godotengine.org/en/stable/classes/class_signal.html>`__, timer\: `SceneTreeTimer <https://docs.godotengine.org/en/stable/classes/class_scenetreetimer.html>`__\ ) |static| :ref:`🔗<class_Async_method_timeout>`

Awaits ``target_signal`` and aborts early if ``timer`` expires first.  Returns ``true`` if the timer fired before the signal, ``false`` if the signal fired in time.

::

    var timer := get_tree().create_timer(5.0)
    if await Async.timeout(my_signal, timer):
        push_error("Timed out waiting for my_signal.")

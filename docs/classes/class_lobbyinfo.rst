LobbyInfo
=========

**Inherits:** `Resource <https://docs.godotengine.org/en/stable/classes/class_resource.html>`__

Plain data describing a single discoverable lobby.

.. rst-class:: classref-introduction-group

Description
-----------

Emitted in batches by `LobbyProvider.lobby_list_updated <https://docs.godotengine.org/en/stable/classes/class_lobbyprovider.html#class_LobbyProvider_signal_lobby_list_updated>`__. UIs render one entry per **LobbyInfo** without caring about the underlying transport.

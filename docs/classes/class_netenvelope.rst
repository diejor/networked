NetEnvelope
===========

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Typed identity wrapper for all debug telemetry messages.

.. rst-class:: classref-introduction-group

Description
-----------

Created once at the source; never mutated downstream. Replaces the ``data.duplicate() + patched["tree_name"] = ...`` pattern. 



Identity is defined by :ref:`source_path<class_NetEnvelope_property_source_path>` (unique within a process) and :ref:`reporter_id<class_NetEnvelope_property_reporter_id>` (differentiates two processes sharing one editor session).

SteamLobbyProvider
==================

**Inherits:** :ref:`LobbyProvider<class_LobbyProvider>` **<** `Node <https://docs.godotengine.org/en/stable/classes/class_node.html>`__

Steam-backed :ref:`LobbyProvider<class_LobbyProvider>` implementation.

.. rst-class:: classref-introduction-group

Description
-----------

Add as a child of :ref:`MultiplayerTree<class_MultiplayerTree>`. Owns the :ref:`SteamWrapper<class_SteamWrapper>`, drives Steam callbacks each frame, and translates Steam signals into provider signals. 



Only one instance may exist per process; duplicates queue themselves for deletion. :ref:`browser_filter_uid<class_SteamLobbyProvider_property_browser_filter_uid>` tags hosted lobbies so the browser only returns lobbies created with the same game id.

::

    @onready var provider := $MultiplayerTree/SteamLobbyProvider
    func _on_create_pressed() -> void:
        provider.lobby_created.connect(_on_lobby_created)
        provider.create_lobby("My Room")

    func _on_lobby_created(_id: int) -> void:
        provider.bind(Netw.tree)

PanelWrapper
============

**Inherits:** `PanelContainer <https://docs.godotengine.org/en/stable/classes/class_panelcontainer.html>`__

Wraps a panel ``Control`` in a titled container with a colored header bar.

.. rst-class:: classref-introduction-group

Description
-----------

Owns the peer-context state (online / remote) and propagates it to the inner panel when the panel implements :ref:`DebugPanel<class_DebugPanel>`. Panels that do not implement :ref:`DebugPanel<class_DebugPanel>` are stored as plain ``Control`` and skipped for hooks.  Double-clicking the title bar fires :ref:`on_maximize_requested<class_PanelWrapper_property_on_maximize_requested>`.

HistoryBuffer
=============

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Pre-allocated ring buffer that stores values keyed by tick number.

.. rst-class:: classref-introduction-group

Description
-----------

Uses power-of-two capacity for fast bitwise indexing and optimized searching to minimize GDScript overhead in hot loops.

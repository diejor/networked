DummyAuthProvider
=================

**Inherits:** :ref:`NetwAuthProvider<class_NetwAuthProvider>` **<** `Resource <https://docs.godotengine.org/en/stable/classes/class_resource.html>`__

Concrete :ref:`NetwAuthProvider<class_NetwAuthProvider>` for testing and development.

.. rst-class:: classref-introduction-group

Description
-----------

Echoes the client-claimed username as the server-authoritative identity. Service is set to ``&"dummy"``.  **Note:** This provider does not perform any real authentication. Use it to verify the auth pipeline wiring without depending on external services.

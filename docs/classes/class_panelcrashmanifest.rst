PanelCrashManifest
==================

**Inherits:** :ref:`DebugPanel<class_DebugPanel>` **<** `VBoxContainer <https://docs.godotengine.org/en/stable/classes/class_vboxcontainer.html>`__

Crash Manifest panel - the Orchestrator Peak.

.. rst-class:: classref-introduction-group

Description
-----------

Displays received ``NetDebugManifest`` entries as an interactive ``Tree``. Selecting a row calls back into :ref:`NetworkedDebuggerUI<class_NetworkedDebuggerUI>` to synchronize the Span Tracer, Matrices, and Components panels.

TPComponent
===========

**Inherits:** :ref:`NetwComponent<class_NetwComponent>` **<** `Node <https://docs.godotengine.org/en/stable/classes/class_node.html>`__

Cross-scene teleportation for a player-owned entity.

.. rst-class:: classref-introduction-group

Description
-----------

:ref:`teleport()<class_TPComponent_method_teleport>` returns a :ref:`TPComponent.TeleportPromise<class_TPComponent_TeleportPromise>` that survives node destruction, so `await <https://docs.godotengine.org/en/stable/classes/class_tpcomponent.html#class_TPComponent_operator_await>`__ is safe across the delete+respawn cycle. Requires a :ref:`TPLayerAPI<class_TPLayerAPI>` in the destination scene for transition animations. 

::

    var tp := %TPComponent.teleport(target_scene)
    await tp.completed

MoveInputComponent
==================

**Inherits:** :ref:`InputComponent<class_InputComponent>` **<** :ref:`NetwComponent<class_NetwComponent>` **<** `Node <https://docs.godotengine.org/en/stable/classes/class_node.html>`__

Concrete :ref:`InputComponent<class_InputComponent>` that tracks four directional movement actions and sprint.

.. rst-class:: classref-introduction-group

Description
-----------

Each action name is configurable via the inspector.  :ref:`direction<class_MoveInputComponent_property_direction>` and :ref:`sprinting<class_MoveInputComponent_property_sprinting>` are updated every physics frame and can be read directly.

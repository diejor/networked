SynchronizersCache
==================

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Cached lookup helpers for ``MultiplayerSynchronizer`` nodes.

.. rst-class:: classref-introduction-group

Description
-----------

Tree traversals are expensive; results are stored on the target node's metadata under ``"cached_synchronizers"`` and reused until explicitly invalidated.

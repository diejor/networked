CheckpointToken
===============

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Lightweight causal link for the Networked span tracing system.

.. rst-class:: classref-introduction-group

Description
-----------

A **CheckpointToken** captures a point in time within a :ref:`NetSpan<class_NetSpan>` so that a subsequent, causally-related span can declare an explicit ``follows_from`` relationship - without touching production node metadata. 



\ **Usage:**\ 

::

    var token := write_span.checkpoint("cache_written")
    func load_data(token: CheckpointToken = null) -> void:
        var span := Netw.dbg.span(self, "cache_read", {}, token)

ProxySynchronizer
=================

**Inherits:** `MultiplayerSynchronizer <https://docs.godotengine.org/en/stable/classes/class_multiplayersynchronizer.html>`__

**Inherited By:** :ref:`SaveComponent<class_SaveComponent>`, :ref:`TickAwareSynchronizer<class_TickAwareSynchronizer>`

``MultiplayerSynchronizer`` that virtualizes arbitrary node properties behind a stable name map.

.. rst-class:: classref-introduction-group

Description
-----------

Subclasses call :ref:`register_property()<class_ProxySynchronizer_method_register_property>` to create virtual names and :ref:`finalize()<class_ProxySynchronizer_method_finalize>` to apply the config. `_read_property() <https://docs.godotengine.org/en/stable/classes/class_proxysynchronizer.html#class_ProxySynchronizer_private_method__read_property>`__ and `_write_property() <https://docs.godotengine.org/en/stable/classes/class_proxysynchronizer.html#class_ProxySynchronizer_private_method__write_property>`__ redirect reads and writes through those names. 

::

    func _ready() -> void:
        register_property(&"velocity", NodePath("Character:velocity"))
        register_property(&"health",   NodePath("Character:health"))
        finalize()

|void| **finalize**\ (\ ) :ref:`🔗<class_ProxySynchronizer_method_finalize>`

Applies the built config to `MultiplayerSynchronizer.replication_config <https://docs.godotengine.org/en/stable/classes/class_multiplayersynchronizer.html#class_MultiplayerSynchronizer_property_replication_config>`__. Converts Inspector-set paths to virtual names automatically.  Pivots `root_path <https://docs.godotengine.org/en/stable/classes/class_proxysynchronizer.html#class_ProxySynchronizer_property_root_path>`__ to ``"."`` so Godot routes replication through `_get() <https://docs.godotengine.org/en/stable/classes/class_proxysynchronizer.html#class_ProxySynchronizer_private_method__get>`__ and `_set() <https://docs.godotengine.org/en/stable/classes/class_proxysynchronizer.html#class_ProxySynchronizer_private_method__set>`__.

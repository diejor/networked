NetwAuthProvider
================

**Inherits:** `Resource <https://docs.godotengine.org/en/stable/classes/class_resource.html>`__

**Inherited By:** :ref:`DummyAuthProvider<class_DummyAuthProvider>`

Abstract base class for authentication providers.

.. rst-class:: classref-introduction-group

Description
-----------

Three lifecycle hooks called by :ref:`MultiplayerTree<class_MultiplayerTree>`: 



\ **Client side:** 

1. `_prepare() <https://docs.godotengine.org/en/stable/classes/class_netwauthprovider.html#class_NetwAuthProvider_private_method__prepare>`__ -- before transport opens. Populates :ref:`JoinPayload<class_JoinPayload>` with metadata. Returns `OK <https://docs.godotengine.org/en/stable/classes/class_netwauthprovider.html#class_NetwAuthProvider_constant_OK>`__ to proceed, or an Error to abort :ref:`MultiplayerTree.connect_player()<class_MultiplayerTree_method_connect_player>`. 

2. `_get_credentials() <https://docs.godotengine.org/en/stable/classes/class_netwauthprovider.html#class_NetwAuthProvider_private_method__get_credentials>`__ -- during Godot's auth phase. Returns proof bytes sent via `SceneMultiplayer.send_auth() <https://docs.godotengine.org/en/stable/classes/class_scenemultiplayer.html#class_SceneMultiplayer_method_send_auth>`__.  



\ **Server side:** 

3. `_authenticate() <https://docs.godotengine.org/en/stable/classes/class_netwauthprovider.html#class_NetwAuthProvider_private_method__authenticate>`__ -- when `SceneMultiplayer.auth_callback <https://docs.godotengine.org/en/stable/classes/class_scenemultiplayer.html#class_SceneMultiplayer_property_auth_callback>`__ fires. Validates proof bytes. Returns :ref:`NetwIdentity<class_NetwIdentity>` to accept, ``null`` to reject. Set :ref:`rejection_reason<class_NetwAuthProvider_property_rejection_reason>` on failure so the server can communicate why.  



\ **Listen-server host:** 

4. `_get_host_identity() <https://docs.godotengine.org/en/stable/classes/class_netwauthprovider.html#class_NetwAuthProvider_private_method__get_host_identity>`__ -- after :ref:`MultiplayerTree.host()<class_MultiplayerTree_method_host>` for peer 1. Godot's auth callback never fires for the local host, so this provides a separate path. Returns ``null`` to skip host auth.

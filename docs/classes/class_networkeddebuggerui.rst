NetworkedDebuggerUI
===================

**Inherits:** `VBoxContainer <https://docs.godotengine.org/en/stable/classes/class_vboxcontainer.html>`__

Main ``"Networked"`` tab in the editor debugger.

.. rst-class:: classref-introduction-group

Description
-----------

A :ref:`DebuggerSession<class_DebuggerSession>` injected by :ref:`NetworkedDebuggerPlugin<class_NetworkedDebuggerPlugin>` owns all data. This class reacts to session signals and drives the layout.  Checking a child creates a :ref:`PanelWrapper<class_PanelWrapper>` in the right grid and calls `populate() <https://docs.godotengine.org/en/stable/classes/class_networkeddebuggerui.html#class_NetworkedDebuggerUI_method_populate>`__ on the panel immediately with existing buffered data.  Right grid: column count auto-adjusts to ``ceil(sqrt(active_count))``. 



Double-clicking a :ref:`PanelWrapper<class_PanelWrapper>` title bar maximizes it; double-clicking again restores.

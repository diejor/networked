DebugPanel
==========

**Inherits:** `VBoxContainer <https://docs.godotengine.org/en/stable/classes/class_vboxcontainer.html>`__

**Inherited By:** :ref:`PanelCrashManifest<class_PanelCrashManifest>`, :ref:`PanelSpanTracer<class_PanelSpanTracer>`, :ref:`PanelTopology<class_PanelTopology>`

Base class for all Networked Debugger panel controls.

.. rst-class:: classref-introduction-group

Description
-----------

Defines the interface contract that :ref:`PanelWrapper<class_PanelWrapper>` relies on for peer-context hooks and data lifecycle. Panels that do not extend this are treated as opaque ``Control`` nodes by the wrapper - they simply don't participate in the hook protocol.

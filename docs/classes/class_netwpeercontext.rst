NetwPeerContext
===============

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

A generic per-peer storage container tied to a :ref:`MultiplayerTree<class_MultiplayerTree>` session.

.. rst-class:: classref-introduction-group

Description
-----------

Components register their own typed ``Bucket`` inner classes here rather than using static variables. Each bucket type is keyed by its inner class object, so **NetwPeerContext** never imports or references consumer types. 





::

    # Example usage:
    var bucket = peer_context.get_bucket(MyComponent.Bucket)
    bucket.some_state = 123

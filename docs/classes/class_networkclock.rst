NetworkClock
============

**Inherits:** `Node <https://docs.godotengine.org/en/stable/classes/class_node.html>`__

Synchronises simulation time between server and clients with drift and stall protection.

.. rst-class:: classref-introduction-group

Description
-----------

The **NetworkClock** provides a stable, tick-based time source required for deterministic simulation and smooth visual interpolation. It handles RTT smoothing, clock drift correction, and frame stall detection to prevent "spiral of death" scenarios. 



\ **Usage:**\ 

::

    # The clock registers itself automatically on the MultiplayerTree.
    # Access it from any node via:
    var clock = NetworkClock.for_node(self)

    # Connect to the simulation loop:
    clock.on_tick.connect(func(delta, tick):
        _simulate_physics(tick)
    )

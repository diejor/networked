WebRTCBackend
=============

**Inherits:** :ref:`BackendPeer<class_BackendPeer>` **<** `Resource <https://docs.godotengine.org/en/stable/classes/class_resource.html>`__

:ref:`BackendPeer<class_BackendPeer>` implementation that uses WebRTC for peer-to-peer NAT traversal.

.. rst-class:: classref-introduction-group

Description
-----------

Peers discover each other through WebTorrent-compatible tracker servers using a shared ``info_hash`` room ID. On `host() <https://docs.godotengine.org/en/stable/classes/class_webrtcbackend.html#class_WebRTCBackend_method_host>`__ the hash is copied to the clipboard; clients pass that hash to `join() <https://docs.godotengine.org/en/stable/classes/class_webrtcbackend.html#class_WebRTCBackend_method_join>`__.

NetwComponent
=============

**Inherits:** `Node <https://docs.godotengine.org/en/stable/classes/class_node.html>`__

**Inherited By:** :ref:`InputComponent<class_InputComponent>`, :ref:`TickInterpolator<class_TickInterpolator>`, :ref:`TPComponent<class_TPComponent>`

Base class for all networked addon components.

.. rst-class:: classref-introduction-group

Description
-----------

Provides instance-method access to session services via :ref:`get_context()<class_NetwComponent_method_get_context>`, which returns a :ref:`NetwContext<class_NetwContext>` safe across node renames.

NetwHandle
==========

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Instance-method proxy for :ref:`NetwDbg<class_NetwDbg>` bound to a specific ``Object``.

.. rst-class:: classref-introduction-group

Description
-----------

Captures a weak reference to the context object. If the object is freed, all calls become silent no-ops.

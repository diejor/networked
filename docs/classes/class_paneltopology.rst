PanelTopology
=============

**Inherits:** :ref:`DebugPanel<class_DebugPanel>` **<** `VBoxContainer <https://docs.godotengine.org/en/stable/classes/class_vboxcontainer.html>`__

Topology panel - shows synchronizer tree and identity for one player peer.

.. rst-class:: classref-introduction-group

Description
-----------

Displays the latest :ref:`NetTopologySnapshot<class_NetTopologySnapshot>` received from the reporter. Topology is current-state (not time-series), so :ref:`populate()<class_PanelTopology_method_populate>` shows only the most recent entry and :ref:`on_new_entry()<class_PanelTopology_method_on_new_entry>` replaces rather than appends.  Layout (top to bottom): ``cache banner``      - hidden unless snapshot is stale identity rows       - username / peer_id, scene / mode node button         - opens the node in the editor scene inspector synchronizer tree   - Name / Mode / Flags / Source columns

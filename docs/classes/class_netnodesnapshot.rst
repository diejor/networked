NetNodeSnapshot
===============

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Typed snapshot of a networked node's debug state at a point in time.

.. rst-class:: classref-introduction-group

Description
-----------

Created via :ref:`from_node()<class_NetNodeSnapshot_method_from_node>` and attached to :ref:`NetTopologyManifest<class_NetTopologyManifest>` so the editor panel can display synchronized property values alongside validation errors. 



Production nodes may contribute additional state by implementing: 



::

    func _get_net_debug_state() -> Dictionary:
        return { "health": health, "state": state_machine.current }

\ 

The return value must contain only basic serializable types (``String``, ``int``, ``float``, ``bool``, ``Array``, ``Dictionary``). Object references are stripped with a warning.

NetwBackend
===========

**Inherits:** `Resource <https://docs.godotengine.org/en/stable/classes/class_resource.html>`__

**Inherited By:** :ref:`DictionaryBackend<class_DictionaryBackend>`, :ref:`FileSystemBackend<class_FileSystemBackend>`

Abstract storage contract for :ref:`NetwDatabase<class_NetwDatabase>` backends.

.. rst-class:: classref-introduction-group

Description
-----------

Extend this resource to implement a concrete storage strategy (e.g. filesystem, SQLite, Postgres). Override all methods to provide the storage implementation. :ref:`NetwDatabase<class_NetwDatabase>` calls these after schema validation.  All table names and record IDs are ``StringName`` for cheap equality checks. Data payloads are plain ``Dictionary`` keyed by column ``StringName``.

BackendPeer
===========

**Inherits:** `Resource <https://docs.godotengine.org/en/stable/classes/class_resource.html>`__

**Inherited By:** :ref:`ENetBackend<class_ENetBackend>`, :ref:`LocalLoopbackBackend<class_LocalLoopbackBackend>`, :ref:`TubeBackend<class_TubeBackend>`, :ref:`WebRTCBackend<class_WebRTCBackend>`, :ref:`WebRTCLoopbackBackend<class_WebRTCLoopbackBackend>`, :ref:`WebSocketBackend<class_WebSocketBackend>`

Abstract base resource for network transports used by :ref:`MultiplayerTree<class_MultiplayerTree>`.

.. rst-class:: classref-introduction-group

Description
-----------

Subclass this to implement a new transport (ENet, WebSocket, WebRTC, etc.). Override :ref:`create_host_peer()<class_BackendPeer_method_create_host_peer>` and :ref:`create_join_peer()<class_BackendPeer_method_create_join_peer>` to produce a ``MultiplayerPeer``; the :ref:`MultiplayerTree<class_MultiplayerTree>` owns the ``SceneMultiplayer`` and assigns the returned peer onto it.

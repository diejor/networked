MultiplayerScene
================

**Inherits:** `Node <https://docs.godotengine.org/en/stable/classes/class_node.html>`__

Container node representing a single game scene (server or client variant).

.. rst-class:: classref-introduction-group

Description
-----------

Created by :ref:`MultiplayerSceneManager<class_MultiplayerSceneManager>` via its spawn function. Holds the instantiated level scene and wires up spawn/despawn signals to the :ref:`SceneSynchronizer<class_SceneSynchronizer>`.

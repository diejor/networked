Use when the interpolated property on the owner has a different name than the property on the visual root.

Controls the "softness" or "floatiness" of the interpolation.

- ``> 0.0``: Adds exponential smoothing, making motion feel heavier/fluid but adding visual lag.

|void| **snap_property**\ (\ property\: `StringName <https://docs.godotengine.org/en/stable/classes/class_stringname.html>`__, value\: `Variant <https://docs.godotengine.org/en/stable/classes/class_variant.html>`__\ ) :ref:`🔗<class_TickInterpolator_method_snap_property>`

Instantly snaps ``property`` to ``value``, bypassing interpolation.

::

    # Snap a base property
    interpolator.snap_property(&"position", Vector2(200, 100))

    # Snap a nested child property (must match the exact key in property_modes)
    interpolator.snap_property(&"Sprite2D:position", Vector2.ZERO)

NetwDatabase.TableRepository
============================

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

A typed read/write interface for a single database table.

.. rst-class:: classref-introduction-group

Description
-----------

Obtain a repository from :ref:`NetwDatabase<class_NetwDatabase>` via :ref:`NetwDatabase.table()<class_NetwDatabase_method_table>`. Repositories surface CRUD operations and return hydrated :ref:`Entity<class_Entity>` instances, keeping all I/O knowledge inside the persistence layer. 

::

    # Fetch an entity and bind it to a SaveComponent:
    var entity := db.table(&"players").fetch(username)
    if entity:
        save_comp.bound_entity = entity

    # Write changes back to the database:
    var err := db.table(&"players").put(username, save_comp.bound_entity)

    # Delete a record:
    db.table(&"players").delete(username)

    # Fetch every entity in the table:
    for entity in db.table(&"players").fetch_all():
        print(entity.get_value(&"score"))

\ **Custom entity classes:** register an entity script via :ref:`NetwDatabase.declare_table()<class_NetwDatabase_method_declare_table>` so :ref:`fetch()<class_NetwDatabase_TableRepository_method_fetch>` returns a typed subclass rather than a :ref:`DictionaryEntity<class_DictionaryEntity>`:

::

    db.declare_table(&"rocks", [&"health", &"position"], RockEntity)
    var rock: RockEntity = db.table(&"rocks").fetch(&"rock_1")

Fetches the record for ``id`` and returns it as a hydrated :ref:`Entity<class_Entity>`.  Returns ``null`` when the record does not exist, the table has no registered schema, or a schema-mismatch policy blocks the load. The caller assigns the entity to the component:

::

    save_comp.bound_entity = db.table(&"players").fetch(username)

WebRTCLoopbackBackend
=====================

**Inherits:** :ref:`BackendPeer<class_BackendPeer>` **<** `Resource <https://docs.godotengine.org/en/stable/classes/class_resource.html>`__

:ref:`BackendPeer<class_BackendPeer>` that routes packets through an in-process :ref:`WebRTCLoopbackSession<class_WebRTCLoopbackSession>`.

.. rst-class:: classref-introduction-group

Description
-----------

Uses the full WebRTC offer/answer/ICE handshake entirely in memory, so it works in web exports without real WebRTC sockets.

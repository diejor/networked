AsyncMutex
==========

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Coroutine-safe mutual exclusion primitive for GDScript ``await`` contexts.

.. rst-class:: classref-introduction-group

Description
-----------

Unlike a real OS mutex, this yields via ``await`` instead of blocking the thread.

::

    var mutex := AsyncMutex.new()
    await mutex.lock()
    # critical section
    mutex.unlock()

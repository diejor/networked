PanelSpanTracer
===============

**Inherits:** :ref:`DebugPanel<class_DebugPanel>` **<** `VBoxContainer <https://docs.godotengine.org/en/stable/classes/class_vboxcontainer.html>`__

Span Tracer panel.

.. rst-class:: classref-introduction-group

Description
-----------

Shows Span rows opened via ``Netw.dbg`` by the addon's own systems. They have an explicit lifecycle indicator: (o) open (yellow), ``V`` closed (green), ``X`` failed (red). Each step is a child row with elapsed-ms timing.  

\ **Breakpoint gutter** (:ref:`COL_BP<class_PanelSpanTracer_constant_COL_BP>`): click the breakpoint icon on any row that has caller info to toggle a GDScript breakpoint at that call site. The icon stays in sync with the script editor via :ref:`sync_breakpoint()<class_PanelSpanTracer_method_sync_breakpoint>` / :ref:`sync_breakpoints_cleared()<class_PanelSpanTracer_method_sync_breakpoints_cleared>`, driven by `NetworkedDebuggerPlugin._breakpoint_set_in_tree() <https://docs.godotengine.org/en/stable/classes/class_networkeddebuggerplugin.html#class_NetworkedDebuggerPlugin_private_method__breakpoint_set_in_tree>`__.  

\ **Jump to source**: double-click any row (or press Enter while selected) to open the script at the call site in the editor.

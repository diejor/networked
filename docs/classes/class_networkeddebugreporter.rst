NetworkedDebugReporter
======================

**Inherits:** `Node <https://docs.godotengine.org/en/stable/classes/class_node.html>`__

Game-side debug telemetry reporter for the Networked debugger plugin.

.. rst-class:: classref-introduction-group

Description
-----------

This is a singleton (Autoload) node that collects telemetry from all active :ref:`MultiplayerTree<class_MultiplayerTree>` instances in the process and forwards them to the editor. 



The bootstrap only creates this node when reporter policy allows it.

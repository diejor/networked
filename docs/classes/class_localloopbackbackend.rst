LocalLoopbackBackend
====================

**Inherits:** :ref:`BackendPeer<class_BackendPeer>` **<** `Resource <https://docs.godotengine.org/en/stable/classes/class_resource.html>`__

:ref:`BackendPeer<class_BackendPeer>` that routes packets through an in-process :ref:`LocalLoopbackSession<class_LocalLoopbackSession>`.

.. rst-class:: classref-introduction-group

Description
-----------

Used automatically by :ref:`MultiplayerTree<class_MultiplayerTree>` when running on the web with a non-WebRTC backend, ensuring a fast, allocation-free loopback without any real network sockets.

NetwEntity
==========

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Bundles spawn lifecycle signals, identity tracking, and property contributions for one entity root.

.. rst-class:: classref-introduction-group

Description
-----------

A **NetwEntity** is attached as metadata on the owner node that hosts :ref:`SpawnerComponent<class_SpawnerComponent>` and :ref:`SaveComponent<class_SaveComponent>`. Access it from any node inside the entity with ``Netw.ctx(self).entity``.  



\ :ref:`entity_id<class_NetwEntity_property_entity_id>` is the display / save / debug label for the entity, for example a username. :ref:`peer_id<class_NetwEntity_property_peer_id>` is the joined peer this entity represents, or ``0`` for world entities.  



The spawn phase (driven by :ref:`SpawnerComponent<class_SpawnerComponent>`) emits `spawning <https://docs.godotengine.org/en/stable/classes/class_netwentity.html#class_NetwEntity_signal_spawning>`__ before scene registration and `spawned <https://docs.godotengine.org/en/stable/classes/class_netwentity.html#class_NetwEntity_signal_spawned>`__ after. Siblings use :ref:`contribute_spawn_property()<class_NetwEntity_method_contribute_spawn_property>` from `Node.NOTIFICATION_PARENTED <https://docs.godotengine.org/en/stable/classes/class_node.html#class_Node_constant_NOTIFICATION_PARENTED>`__ to inject spawn-synced paths.

::

    func _notification(what: int) -> void:
        if what == NOTIFICATION_PARENTED:
            var entity := Netw.ctx(self).entity
            entity.contribute_spawn_property(NodePath("..:health"))
            entity.spawning.connect(_on_spawning)

    func _on_spawning() -> void:
        if multiplayer.is_server():
            restore_saved_state()

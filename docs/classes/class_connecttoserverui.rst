ConnectToServerUI
=================

**Inherits:** `MarginContainer <https://docs.godotengine.org/en/stable/classes/class_margincontainer.html>`__

Simple form UI with server IP and username fields that emits connection data.

.. rst-class:: classref-introduction-group

Description
-----------

Place inside a ``connect_server_ui.gd`` scene. Connect `connect_player <https://docs.godotengine.org/en/stable/classes/class_connecttoserverui.html#class_ConnectToServerUI_signal_connect_player>`__ to the parent overlay's forwarding signal.

NetwDbg
=======

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

**Inherited By:** :ref:`NetwDbgNoop<class_NetwDbgNoop>`

Access point for Networked debug functionality.

.. rst-class:: classref-introduction-group

Description
-----------

``Netw.dbg`` owns logging, span creation, trace sink policy, reporter discovery, and component handles.

::

    Netw.dbg.info("Player %s connected", [player_name])
    var span := Netw.dbg.peer_span(self, "spawn", peers)

\ 



Pass log arguments as an ``Array`` to defer string formatting. Guard hot paths with :ref:`is_level_active()<class_NetwDbg_method_is_level_active>` before constructing expensive arguments. 



Pass a lambda to :ref:`warn()<class_NetwDbg_method_warn>` or :ref:`error()<class_NetwDbg_method_error>` to preserve editor jump-click behaviour.

::

    Netw.dbg.error("Critical failure", [], func(m): push_error(m))

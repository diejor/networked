PanelDataAdapter
================

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

**Inherited By:** :ref:`CrashAdapter<class_CrashAdapter>`, :ref:`SpanAdapter<class_SpanAdapter>`, :ref:`TopologyAdapter<class_TopologyAdapter>`

Base class for all per-peer panel data adapters.

.. rst-class:: classref-introduction-group

Description
-----------

Each adapter owns a ring buffer for one peer+panel combination. Subclasses override :ref:`feed()<class_PanelDataAdapter_method_feed>` to append entries and emit `data_changed <https://docs.godotengine.org/en/stable/classes/class_paneldataadapter.html#class_PanelDataAdapter_signal_data_changed>`__.

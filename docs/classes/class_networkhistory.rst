NetworkHistory
==============

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Stores per-node state and input history for rollback.

.. rst-class:: classref-introduction-group

Description
-----------

Snapshots are keyed by tick so a rollback loop can restore any past tick and replay from there.

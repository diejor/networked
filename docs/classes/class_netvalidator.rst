NetValidator
============

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

**Inherited By:** :ref:`TopologyNetValidator<class_TopologyNetValidator>`

Base class for the Networked phased validation engine.

.. rst-class:: classref-introduction-group

Description
-----------

Validators plug into `NetworkedDebugReporter._execute_validators() <https://docs.godotengine.org/en/stable/classes/class_networkeddebugreporter.html#class_NetworkedDebugReporter_private_method__execute_validators>`__ and are invoked in phase order. If any validator in a phase returns errors, later phases are skipped — preventing cascading failures (e.g. heuristic checks on a structurally broken node). 



Override :ref:`execute()<class_NetValidator_method_execute>` in subclasses to implement detection logic. Return an empty array to indicate the check passed. 



\ **Note:** The validator pattern is scoped to topology-style checks that return plain error strings. Detection logic that requires structured output (race data, zombie scans) lives in dedicated reporters, not validators.

NetwLog
=======

**Inherits:** `Object <https://docs.godotengine.org/en/stable/classes/class_object.html>`__

Static logging utility for the networked addon with per-module level overrides.

.. rst-class:: classref-introduction-group

Description
-----------

Log calls resolve a per-module level by walking the dot-separated module hierarchy. At runtime the first log call lazily detects the addon root and loads the active :ref:`NetwLogSettings<class_NetwLogSettings>` profile from ``ProjectSettings``.

::

    Netw.dbg.info(self, "Player spawned: %s", [username])
    Netw.dbg.warn(self, "Connection attempt failed, retrying...")

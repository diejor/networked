NetwLogSettings
===============

**Inherits:** `Resource <https://docs.godotengine.org/en/stable/classes/class_resource.html>`__

Serializable logging profile resource consumed by :ref:`NetwLog<class_NetwLog>`.

.. rst-class:: classref-introduction-group

Description
-----------

Create instances of this resource as ``.tres`` files and assign one as the active profile in **Project Settings > networked/logging/active_profile**. The ``global_level`` field uses ``int`` rather than :ref:`Level<enum_NetwLog_Level>` to avoid a circular dependency during resource parsing (INHERIT=-1 … NONE=5).

NetTopologySnapshot
===================

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Debug event carrying the full synchronizer topology for one player node.

.. rst-class:: classref-introduction-group

Description
-----------

Sent by :ref:`NetworkedDebugReporter<class_NetworkedDebugReporter>` on every player spawn. 



The topology panel treats the latest entry as ground truth - it always shows the most recent snapshot, not a time-series.

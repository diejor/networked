NetworkedDebuggerPlugin
=======================

**Inherits:** `EditorDebuggerPlugin <https://docs.godotengine.org/en/stable/classes/class_editordebuggerplugin.html>`__

Editor-side plugin that registers the "Networked" debugger tab.

.. rst-class:: classref-introduction-group

Description
-----------

Handles the ``EditorDebuggerPlugin`` lifecycle: creates a :ref:`DebuggerSession<class_DebuggerSession>` and :ref:`NetworkedDebuggerUI<class_NetworkedDebuggerUI>` per session. All incoming game messages are routed to :ref:`DebuggerSession.receive()<class_DebuggerSession_method_receive>`; the UI reacts to session signals rather than handling messages directly.

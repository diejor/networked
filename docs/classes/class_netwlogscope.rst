NetwLogScope
============

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Restores a :ref:`NetwLog<class_NetwLog>` override when :ref:`close()<class_NetwLogScope_method_close>` is called.

.. rst-class:: classref-introduction-group

Description
-----------

Keep the returned scope in a variable for as long as the override should remain active.

::

    var scope := NetwLog.scoped("trace")
    # ...
    scope.close()

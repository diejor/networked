NetwServices
============

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Service locator facade over :ref:`MultiplayerTree<class_MultiplayerTree>`.

.. rst-class:: classref-introduction-group

Description
-----------

Provides access to backend systems such as :ref:`MultiplayerSceneManager<class_MultiplayerSceneManager>` and :ref:`NetworkClock<class_NetworkClock>` without exposing the concrete :ref:`MultiplayerTree<class_MultiplayerTree>` class to component code. 



\ **Custom Services**  Register your own session services with :ref:`register()<class_NetwServices_method_register>` and :ref:`unregister()<class_NetwServices_method_unregister>`:

::

    class_name MyLobbyManager
    extends Node

    func _enter_tree() -> void:
        NetwServices.register(self)

    func _exit_tree() -> void:
        NetwServices.unregister(self)

\ Retrieve from any component via :ref:`NetwContext.services<class_NetwContext_property_services>`:

::

    var ctx := get_context()
    var lobby: MyLobbyManager = \
            ctx.services.get_service(MyLobbyManager)

\ 



Obtain via :ref:`NetwContext.services<class_NetwContext_property_services>` rather than constructing directly. Holds a ``WeakRef`` so cached references do not keep the tree alive.

Registers ``service`` as a session service.  Walks ``service``'s ancestor chain to find the owning :ref:`MultiplayerTree<class_MultiplayerTree>`, then calls :ref:`MultiplayerTree.register_service()<class_MultiplayerTree_method_register_service>`. 



Call this in ``_enter_tree``:

::

    func _enter_tree() -> void:
        var mt := NetwServices.register(self)

\ If ``type`` is ``null``, the service's script class is used as the registration key. 



Returns the owning :ref:`MultiplayerTree<class_MultiplayerTree>`, or ``null`` if ``service`` is not a descendant of one.

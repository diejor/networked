NetwDatabase
============

**Inherits:** `Resource <https://docs.godotengine.org/en/stable/classes/class_resource.html>`__

Schema registry and API surface for the networked persistence layer.

.. rst-class:: classref-introduction-group

Description
-----------

Save this resource as a ``.tres`` file and assign it to the :ref:`SaveComponent.database<class_SaveComponent_property_database>` export on any entity that should participate in the database.  



\ **Table access:** retrieve a :ref:`NetwDatabase.TableRepository<class_NetwDatabase_TableRepository>` for a named table with :ref:`table()<class_NetwDatabase_method_table>`, then fetch, put, or delete :ref:`Entity<class_Entity>` records:

::

    # Fetch a player entity:
    var entity := db.table(&"players").fetch(username)

    # Property-style access also works (autocomplete via _get_property_list):
    var entity := db.players.fetch(username)

\ **Typed tables:** register an entity subclass so `TableRepository() <https://docs.godotengine.org/en/stable/classes/class_netwdatabase.html#class_NetwDatabase_method_TableRepository>`__ returns the right type automatically:

::

    db.declare_table(&"rocks", [&"health", &"position"], RockEntity)
    var rock: RockEntity = db.table(&"rocks").fetch(&"rock_1")

\ **Transaction API:** batch writes via a closure to guarantee the commit always runs:

::

    db.transaction(func(tx: NetwDatabase.TransactionContext):
        tx.queue_upsert(&"rocks", &"rock_1", {&"health": 50})
        tx.queue_upsert(&"rocks", &"rock_2", {&"health": 75})
    )

\ **Schema mismatch:** when a loaded record contains columns not present in the current schema (e.g. after a property rename) the :ref:`mismatch_policy<class_NetwDatabase_property_mismatch_policy>` determines what happens — see :ref:`SchemaMismatchPolicy<enum_NetwDatabase_SchemaMismatchPolicy>`.

Declares ``table_name`` with ``columns`` and an optional typed ``entity_script``.  This is the public entry point for build-time or power-user schema declaration. Tables declared here are known before any runtime query, preventing silent schema-mismatch destruction of data.

::

    var rock: RockEntity = db.table(&"rocks").fetch(&"rock_1")

.. _class_NetwDatabase_method_transaction:

.. rst-class:: classref-method

Error **transaction**\ (\ body\: `Callable <https://docs.godotengine.org/en/stable/classes/class_callable.html>`__\ ) :ref:`🔗<class_NetwDatabase_method_transaction>`

Collects upserts inside a ``Callable`` and commits them all at once.  ``body`` receives a :ref:`NetwDatabase.TransactionContext<class_NetwDatabase_TransactionContext>` and should call `TransactionContext() <https://docs.godotengine.org/en/stable/classes/class_netwdatabase.html#class_NetwDatabase_method_TransactionContext>`__ for each record to write. The transaction is committed after ``body`` returns. Returns `OK <https://docs.godotengine.org/en/stable/classes/class_netwdatabase.html#class_NetwDatabase_constant_OK>`__ on success or the first error returned by the backend.

NetRaceDetector
===============

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Static utility for detecting network races in Godot's Multiplayer API.

.. rst-class:: classref-introduction-group

Description
-----------

Specifically identifies "Simplify Path Races" where a property update (``simplify_path``) arrives at a client before the corresponding spawn packet, causing "Node not found" errors.

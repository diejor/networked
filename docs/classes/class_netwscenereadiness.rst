NetwSceneReadiness
==================

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Per-scene readiness gate — tracks which players have confirmed ready.

.. rst-class:: classref-introduction-group

Description
-----------

Obtain via :ref:`NetwScene.create_readiness_gate()<class_NetwScene_method_create_readiness_gate>`. Clients call :ref:`set_ready()<class_NetwSceneReadiness_method_set_ready>`; the server broadcasts the change to all peers.

::

    # Game scene screen (runs on all peers):
    var gate := ctx.scene.create_readiness_gate()
    gate.player_ready_changed.connect(_refresh_ready_ui)
    gate.all_ready.connect(_on_everyone_ready)

    # Player clicks "Ready":
    gate.set_ready(true)

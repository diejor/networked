TPLayerAPI
==========

**Inherits:** `CanvasLayer <https://docs.godotengine.org/en/stable/classes/class_canvaslayer.html>`__

Abstract base class for client-side teleport transition overlays.

.. rst-class:: classref-introduction-group

Description
-----------

Automatically registers itself with :ref:`NetwServices<class_NetwServices>` when added to the tree. Subclasses implement :ref:`teleport_out()<class_TPLayerAPI_method_teleport_out>` (fade/cover outgoing scene) and :ref:`teleport_in()<class_TPLayerAPI_method_teleport_in>` (reveal incoming scene). Both methods are awaitable.

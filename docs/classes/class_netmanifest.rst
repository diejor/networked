NetManifest
===========

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

**Inherited By:** :ref:`NetCppErrorManifest<class_NetCppErrorManifest>`, :ref:`NetRaceManifest<class_NetRaceManifest>`, :ref:`NetTopologyManifest<class_NetTopologyManifest>`, :ref:`NetZombieManifest<class_NetZombieManifest>`

Base class for typed crash manifests crossing the ``EngineDebugger`` boundary.

.. rst-class:: classref-introduction-group

Description
-----------

Holds the fields common to every manifest. Subclasses add trigger-specific fields and call ``super.to_dict()`` when serializing. 



\ **Key invariant:** :ref:`to_dict()<class_NetManifest_method_to_dict>` must produce the exact key set that ``ManifestFormatter.format`` expects. Do not rename keys without updating the formatter.

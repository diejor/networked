InputComponent
==============

**Inherits:** :ref:`NetwComponent<class_NetwComponent>` **<** `Node <https://docs.godotengine.org/en/stable/classes/class_node.html>`__

**Inherited By:** :ref:`MoveInputComponent<class_MoveInputComponent>`

Abstract base component for multiplayer-aware input handling.

.. rst-class:: classref-introduction-group

Description
-----------

Subclass this and implement :ref:`get_inputs()<class_InputComponent_method_get_inputs>` to return the list of action names your component tracks. Processing is automatically disabled on non-authoritative peers.

::

    class_name MoveInputComponent
    extends InputComponent

    @export_custom(PROPERTY_HINT_INPUT_NAME, &"input")
    var move_left: StringName = "move_left"

    func get_inputs() -> Array:
        return [move_left, ...]

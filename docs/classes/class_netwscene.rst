NetwScene
=========

**Inherits:** `RefCounted <https://docs.godotengine.org/en/stable/classes/class_refcounted.html>`__

Scene-scoped facade providing player tracking, lifecycle signals, and server operations.

.. rst-class:: classref-introduction-group

Description
-----------

Access via :ref:`NetwComponent.get_context()<class_NetwComponent_method_get_context>` or :ref:`for_node()<class_NetwScene_method_for_node>`. Holds a ``WeakRef`` to the underlying :ref:`MultiplayerScene<class_MultiplayerScene>` - check :ref:`is_valid()<class_NetwScene_method_is_valid>` before use.

::

    var ctx := get_context()

    # Wait for players then count down
    await ctx.scene.wait_for_players(4)
    var cd := ctx.scene.start_countdown(10)
    await cd.finished
    start_match()

.. _class_NetwScene_method_start_countdown:

.. rst-class:: classref-method

:ref:`NetwSceneCountdown<class_NetwSceneCountdown>` **start_countdown**\ (\ seconds\: `int <https://docs.godotengine.org/en/stable/classes/class_int.html>`__\ ) :ref:`🔗<class_NetwScene_method_start_countdown>`

Starts a server-driven countdown of ``seconds`` seconds.  **Server-only.** Returns a :ref:`NetwSceneCountdown<class_NetwSceneCountdown>` you can ``await``. Clients receive `countdown_started <https://docs.godotengine.org/en/stable/classes/class_netwscene.html#class_NetwScene_signal_countdown_started>`__ followed by `countdown_tick <https://docs.godotengine.org/en/stable/classes/class_netwscene.html#class_NetwScene_signal_countdown_tick>`__ each second, and finally `countdown_finished <https://docs.godotengine.org/en/stable/classes/class_netwscene.html#class_NetwScene_signal_countdown_finished>`__ (or `countdown_cancelled <https://docs.godotengine.org/en/stable/classes/class_netwscene.html#class_NetwScene_signal_countdown_cancelled>`__ if :ref:`cancel_countdown()<class_NetwScene_method_cancel_countdown>` is called first). Any previously running countdown is cancelled automatically.

.. _class_NetwScene_method_create_readiness_gate:

.. rst-class:: classref-method

:ref:`NetwSceneReadiness<class_NetwSceneReadiness>` **create_readiness_gate**\ (\ ) :ref:`🔗<class_NetwScene_method_create_readiness_gate>`

Creates and returns a new :ref:`NetwSceneReadiness<class_NetwSceneReadiness>` gate for this scene.  The gate is pre-populated with all currently connected players (all marked not-ready). Players that join or leave after creation are tracked automatically. Multiple independent gates can be active simultaneously.

ActiveSceneView
===============

**Inherits:** `Control <https://docs.godotengine.org/en/stable/classes/class_control.html>`__

Renders an active ``SubViewport`` scene into the host's root viewport.

.. rst-class:: classref-introduction-group

Description
-----------

Add this as a child of :ref:`MultiplayerTree<class_MultiplayerTree>` in your main scene. On listen-server hosts, this view pulls the local player's current scene directly from the tree and :ref:`MultiplayerSceneManager<class_MultiplayerSceneManager>`, rather than being pushed to by the scene manager. 



Pure clients and dedicated servers do not need this node — pure clients render their scene directly into root, and dedicated servers don't render at all. 



\ **Layout:** defaults to filling its parent rect (PRESET_FULL_RECT). Override anchors after adding it if you want a partial-screen view.

`bool <https://docs.godotengine.org/en/stable/classes/class_bool.html>`__ **auto_resize_target** = ``true`` :ref:`🔗<class_ActiveSceneView_property_auto_resize_target>`

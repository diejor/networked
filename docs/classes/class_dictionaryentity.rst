DictionaryEntity
================

**Inherits:** :ref:`Entity<class_Entity>` **<** :ref:`Serde<class_Serde>` **<** `Resource <https://docs.godotengine.org/en/stable/classes/class_resource.html>`__

:ref:`Entity<class_Entity>` backed by a plain ``Dictionary``.

.. rst-class:: classref-introduction-group

Description
-----------

Stores arbitrary ``Variant`` values keyed by ``StringName``. Round-trips through ``PackedByteArray`` via ``var_to_bytes`` / ``bytes_to_var``. 

::

    var entity := DictionaryEntity.new()
    entity.set_value(&"health", 100)
    entity.set_value(&"position", Vector2(10, 20))

    var bytes := entity.serialize()
    var copy := DictionaryEntity.new()
    copy.deserialize(bytes)
